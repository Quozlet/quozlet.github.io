Introduction

While this website currently does not have much content, it will likely expand over time. I consider it a personal exercise in both design and web technologies, and as such it will probably look horrifying and use technologies I have no excuse to be using. Nevertheless, it will remain both open-source and reasonably tested on modern browsers.

I'm a minimal person, and for that reason this will probably remain very static and lightweight. Right now that means only using Tailwind CSS for help styling.
